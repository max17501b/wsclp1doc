Zone Setup Procedures

If you look at this and think, This is insanely complicated," then relax, ignore it, do something else. This is all optional, only takes a few seconds, only has to be done once, and it can save you time and trouble later.

Zone Setup lets you define three properties for each zone.

Description

Type a short description for the zone, something like "Rose Beds" or "North Lawn". This will appear in the Actvities page, the Activities Log, the Program Editor, and the Settings page. It will make these pages and reports more meaningful.

Priority

Change priorities to reorder zones. You may find you want zones to run in an order different from the natural wiring order.

Wire Color

Why record wire color? If you've ever moved or replaced a sprinkler controller you would know. Figuring out which zone is on which solenoid is on which sprinkler heads can take a while, so save all that accumulated wisdom.

Using the Test button

The test button turns on the zone's solenoid for 60 seconds. This action overrides disabled status, and interrupts any other zone activity.

The purpose of testing a zone might be

  • To determine for sure what solenoid/sprinklers the zone services
  • To see if the zone is drawing excessive current
