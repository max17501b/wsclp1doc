WSC-LP1 User Manual

Operation

Settings

All settings are optional. But tweaking a few things can make the controller easier to use. Also, if you have more than one controller, you must give them different names.

Controller Name

Enter a network name for your controller. Names are alpha-numeric plus the "-" and "_" characters. If the editor turns red while you're typing a name it means you have entered illegal characters.

Zones

Zones are listed in priority order. For each zone, you see a zone status icon, the zone name, the optional zone description, and a setup button.
