WSC-LP1 User Manual

Operation

Schedules

The Schedules page controls the scheduling and details of Programs. A program defines when sprinklers come on, and how long they stay on, and a few other details.

Schedule Summary

First thing on the Schedules page is a schedule summary. This tells you

  • Whether the scheduler is running or suspended.
  • What program is scheduled next, and roughly when it starts.
  • And buttons are provided to suspend or resume scheduling.

On the right of the summary is one of two buttons.

Suspends the schedule
Resumes the schedule

When you suspend the scheduler, it stops any future scheduled activity until you resume scheduling. You might do that for instance if you see rain forecast and you don't expect to need the sprinklers for a few days.

Currently running programs continue to completion. Programs which are due to run are skipped until scheduling is resumed.

To immediately turn off all running programs use the Activities Page.

Program Summaries

A program summary is displayed for each program stored on the controller. The controller comes with a built-in program called "Zone Test" which will be used for illustration. The summary shows the program name (Zone Test), how it is scheduled (manual start) and how long it runs (6 minutes). There are also three buttons associated with each program:

Runs an editor so you can change the program details.
Runs the program right here and now. This is the only way to run manual-start programs.
Deletes the program. There is no "undo". But programs are stupid-easy to recreate so don't freak out.

Create Your Own Programs

Click on the button "Create a New Program" to make your own programs.
