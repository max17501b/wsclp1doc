Settings

Sprinkler Zones

Zone Status
State Symbol Description
Normal Zone is operable. It can be scheduled by program and operated manually.
Isolated The zone has failed at least once. It can still be scheduled by program and operated manually. Regardless of its priority, the zone will be serviced last, after all normal zones have completed. Soak breaks are ignored.
Disabled The zone is disabled. It cannot be activated by program or manually. However it can be tested using the zone setup dialog.
