WSC-LP1 User Manual

Operation

Editing Programs

The Edit page controls the definition and modification of Programs. A program defines when sprinklers come on, and how long they stay on, and a few other details.

Name

Type in a descriptive name for the program.

There may be a bug in WSC-LP1 version 1.0.8 which will cause problems is program names are duplicated. Take care to ensure program names are unique.

Zone Water Time

Type in the watering time (in minutes) for each zone. Times can range from 0 to 120 minutes per zone.

You can enter time values greater than 120 minutes, but these values will be truncated by the controller and limited to 120 minutes.

Schedule Method

The schedule method editor selects one of five methods for scheduling programs.

Every Day The program runs at the same time (defined below) every day.
Odd Days The program runs at the same time (defined below) every other day, on odd-numbered days.
Even Days The program runs at the same time (defined below) every other day, on even-numbered days.
Day Of Week The program runs at the same time (defined below) on specific days of the week (also specified below.)
Manual The program can only be started manually.

Start/End Time

The Start/End Time editor specifies the time-of-day when a program starts or finishes. The editor has two parts: First you choose whether to specify time as a start time, or an end time. "What the hell?" I hear you ask. Start time is obvious. You want the program to start at 08:00 or 16:00 or whatever. It finishes when it's done.

But let's say you live in the land of the water police where watering after 08:00 gets you a big ticket. You have a swell program that finishes at 07:55. But then you notice the petunias look a little parched so you up that zone by 10 minutes and BOOM you get a $200 ticket. Wouldn't it be better to say you don't care when a program starts so long as it ends by 07:55? That's what the "End at" selector does. The controller automatically starts the program so it finishes at or before the end time.

The actual start/end time is local time, 24-hour format. E.g. 2:00PM = 14:00.

Schedule Days

The Schedule Days editor only appears if you have selected the scheduling method "Day Of Week". Pick the days on which you want the program to run. Clicking or tapping a particular day will select or de-select that day.

Soak Breaks

Soak breaks interrupt your program to let water percolate and soak into hard ground, or zones prone to run-off. It works by dividing long water times into shorter times, and rotating between zones.

One soak break divides your program into two equal parts. Two soak breaks make three equal parts, etc. Soak breaks add nothing to the actual watering time for a zone, it just breaks the time up.
